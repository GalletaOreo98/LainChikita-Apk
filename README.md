## Building

### Supported platforms  

| Platforms | Supported | Tested |
| :--- | :--- | :--- |
| Android | Yes | 11, 12 |
| IOS | Yes | No |
| Windows | Yes | W10x64 |
| Linux | Yes | Ubuntu22.04.3x64 |
| Web | No | No |

### Building commands  

1. **cd lain_chikita/**  
     *cd to the project folder*
3. **flutter build apk --release**  
     *build the .apk*
